Componentes de una Computadora

Una computadora es un sistema electrónico capaz de recibir, procesar, almacenar y emitir información mediante la interacción entre sus componentes lógicos (software) y físicos (hardware)

Software

El software es la parte lógica de la computadora. Se refiere al conjunto de programas, aplicaciones y sistemas operativos que permiten que el hardware funcione.

Ejemplos de software:

  • Sistemas operativos: Windows, Linux, macOS
  • Aplicaciones: navegadores web, procesadores de texto, editores de imágenes
  • Juegos y herramientas de desarrollo

Hardware

El hardware es la parte física de la computadora, es decir, todo lo que se puede ver y tocar. Incluye tanto componentes internos como externos.

Hardware Interno

El hardware interno de una computadora son todos los componentes físicos y electrónicos que residen dentro del chasis o gabinete de la máquina. Son las partes tangibles que no están directamente accesibles al usuario sin abrir el equipo, pero que son absolutamente esenciales para que la computadora funcione.

Ejemplos de Hardware Interno:

  • Placa base (motherboard)
  • Procesador (CPU)
  • Memoria RAM
  • Disco duro o SSD
  • Fuente de poder

Hardware Externo:

El hardware externo es el conjunto de dispositivos físicos que se conectan a la computadora desde el exterior y permiten la interacción con el usuario o la expansión de sus funciones. Se clasifican principalmente en dispositivos de entrada y dispositivos de salida.

Dispositivos de Entrada:

  • Teclado
  • Ratón (mouse)
  • Escáner
  • Micrófono
  • Cámara web

Dispositivos de Salida:

  • Monitor o pantalla
  • Impresora
  • Bocinas o altavoces
  • Proyector
  • Cámara web
  • Auriculares

Pasos para Armar una Computadora

  1. Instalar el procesador (CPU) en la placa madre (motherboard)
  2. Colocar la memoria RAM en sus ranuras
  3. Montar la placa madre dentro del gabinete
  4. Conectar la fuente de poder
  5. Instalar el disco duro o SSD
  6. Instalar tarjeta gráfica (si es necesaria)
  7. Conectar todos los cables de energía y datos
  8. Cerrar el gabinete
  9. Conectar periféricos o dispositivos de entrada y salida
  10. Encender la PC e instalar el sistema operativo
